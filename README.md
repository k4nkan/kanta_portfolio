## 📝 **概要 (Overview)**

このポートフォリオサイトは、自分のスキルやプロジェクトを紹介するために作成しました。  
レスポンシブデザインを採用し、どのデバイスでも快適に閲覧できます。

**主な目的:**
- 自己紹介やスキルの可視化
- 過去のプロジェクトや成果物の紹介
- SNSリンクの提供

---

## ✨ **主な機能 (Features)**

- 📄 **自己紹介**
- 🖼️ **ギャラリー**
- 🌐 **アクセス**
  
---


## 🛠️ **使用技術 (Technologies)**

このプロジェクトは以下の技術を使用して構築されています:

- **フレームワーク**: [Next.js](https://nextjs.org/)
- **プログラミング言語**: TypeScript
- **スタイリング**: tailwind CSS
- **デプロイ**: [Vercel](https://vercel.com/)
